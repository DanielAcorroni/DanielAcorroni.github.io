Quem sou eu?

Meu nome é Daniel Acorroni, tenho 22 anos e tenho aprendido a programar desde julho de 2021 na Trybe. Eu já conclui os módulos de fundamentos do desenvolvimento web, front end, back end, e ciência da computação, e trabalho hoje como programador pleno na Seu Cliente Oculto. Tenho conhecimento em linguagens e ferramentas como JavaScript, TypeScript, Node.js, Sequelize, Docker, SQL, MySql, Python, Django, Programação Orientada a Objetos (POO), entre outras. Em meus projetos já trabalhei com deploy no Heroku, configuração de containers Docker em front e back end, queries em um banco de dados SQL já existente, normalização de banco de dados, criação de API's com NODE.js, testes com NODE.js utilizando Mocha, Chai e Sinon, aderência ao padrão REST e ao padrão arquitetural MSC, utilização da biblioteca de ORM sequelize, validação por JSON web token e construção de uma API NODE utilizando o TypeScript e a programação orientada a objetos (POO). Pretendo futuramente estudar inteligência artificial e me tornar um nome relevante dentro da pesquisa na área.
